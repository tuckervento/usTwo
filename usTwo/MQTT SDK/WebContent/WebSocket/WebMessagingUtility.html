Use this utility to send and receive messages from your server. Basic functions can be accessed from the "Basic" tab, use the other tabs for more advanced functions.

The utility keeps a trace of which client interface calls are made and the network flows that travel accross the network. Use the "Connect" tab to to expose a view that allows you to see the trace history or send a copy using email.
